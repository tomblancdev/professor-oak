List all extra learnings you've discovered.

## Arguments
- $ARGUMENTS: [topic] (optional - filter by topic)

## Flow

1. Call `listExtras({ topic: $ARGUMENTS })`
2. Display extra learnings grouped by topic
3. Show which ones have associated Pokemon
4. Show tags/categories

## Display Format
```
Your Extra Discoveries:

Docker (3 extras):
  - Docker Volumes Deep Dive [has Pokemon: Squirtle]
  - Best Practices for Dockerfiles
  - Multi-stage Build Tips [has Pokemon: Wartortle]

Python (2 extras):
  - Async/Await Patterns
  - Decorator Magic [has Pokemon: Alakazam]
```

## Persona
Use neutral/informative tone (no specific persona)
